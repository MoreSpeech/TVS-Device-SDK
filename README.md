# TVS-Device-SDK

TVS Device SDK为客户端提供了腾讯语音唤醒、语音识别、语义识别、语音合成的能力。

#### 注意
* 开发者接入需要申请APP KEY，否则语义识别只有chat领域。请联系腾讯叮当团队。
* github上没有放置唤醒库和唤醒模型。如有需要，请联系腾讯叮当团队。
