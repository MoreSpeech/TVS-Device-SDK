AISDK_API_EXPORTS int aisdkCancelText2SpeechByUserData ( void *  userData)

根据外部传入数据，取消特定TTS请求

警告
取消后， 这一次的tts请求结果不再返回。
参数

返回
0:ok other:fail。 错误码定义见AISDK_ERROR_*常量
AISDK_API_EXPORTS int aisdkText2Speech ( const char *  text,
int  textLen,
void *  userData,
int  len 
)

TTS识别接口， TTS结果通过回调异步回调返回

参数
text需要转语音的文本，以utf8格式编码的文本串，以'\0'结尾
textLen文本长度
userData自定义数据
len自定义数据长度
返回
0:ok, other：fail。 错误码定义见AISDK_ERROR_*常量
