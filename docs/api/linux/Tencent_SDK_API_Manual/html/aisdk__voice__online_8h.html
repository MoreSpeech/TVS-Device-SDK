AISDK_API_EXPORTS int aisdkInputOnlineVoice2TextAudioData ( char *  data,
int  dataLen 
)

输入在线录音数据

参数
data录音数据存储区域。建议长度为4096.
dataLen录音数据长度。
返回
0：ok，other：fail。 错误码定义见AISDK_ERROR_*常量
警告
: 上层应用应当把实时录音数据通过该接口输入到SDK（建议每次输入4096字节长度的音频）。 在调用此接口过程中，会有VAD检测到开始、VAD检测到结束、实时音量、语音识别到结果等 状态的回调。遇到AISDK_CMD_ONLINE_RECO_CANCELED/AISDK_CMD_ONLINE_RECO_TIME OUT/AISDK_CMD_ONLINE_RECO_ERROR/AISDK_CMD_ONLINE_RECO_RESULT时，停止调用 该函数。
AISDK_API_EXPORTS int aisdkStartOnlineVoice2Text ( void *  userData,
int  len,
int  flags 
)

开始一次语音识别流程

参数
userData自定义数据指针。callback时带回。
len自定义数据长度。
flags控制标志，参考AISDK_FLAG_ONLINE_RECO_*常量定义，支持多flag或运算。如果不设置，传0即可。
返回
0：ok，other：fail。 错误码定义见AISDK_ERROR_*常量
AISDK_API_EXPORTS int aisdkStopOnlineVoice2Text ( )

停止本次语音识别，获取语音识别结果。

警告
停止后，不会中断语音识别，SDK会将现在已经接收到的音频发送到后台请求语音识别结果。
返回
0：ok，other：fail。 错误码定义见AISDK_ERROR_*常量
