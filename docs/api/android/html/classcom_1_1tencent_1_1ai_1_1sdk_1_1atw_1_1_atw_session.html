◆ appendAudioData()

synchronized int com.tencent.ai.sdk.atw.AtwSession.appendAudioData ( byte []  audioBuffer,
int  bufferLength 
)

输入录音数据

注解
输入实时的录音数据,支持输入16K采样率、S16-LE、单声道的PCM录音,只能在一个线程中调用此接口
警告
必须处于唤醒启动状态下才能输入音频数据。
参数
audioBuffer录音缓存数组，建议每次输入4096字节。
bufferLength录音缓存数组长度
返回
ISSErrors.ISS_SUCCESS:成功，但写入的录音的大小可能为0，other:fail。错误码见ISSErrors内常量
