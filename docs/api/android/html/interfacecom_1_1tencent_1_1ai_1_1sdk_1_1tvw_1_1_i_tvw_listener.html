唤醒初始化结果 参数: state - 初始化状态， true or false errId - 错误码 state=true时, errId可忽略 state=false时， errId取值为BIND_FAILED 若errId=BIND_FAILED，绑定唤醒服务失败。失败原因： MvwSession构造函数中传入的Context对象无法绑定服务。 请参见：http://code.google.com/p/android/issues/detail?id=2483#c2 即：TabSpec Context无法绑定服务。 建议在MvwSession构造函数中直接传入Context.getApplicationContext()
