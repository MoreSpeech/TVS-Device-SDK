typedef void(AISDK_CALL_CONV * AISDK_CALLBACK)(int cmd, char *data, int dataLen, void *userData, int userDataLen, void *extraData, int extraDataLen)

设置回调函数原型

参数
cmd当前的指令，指示参数data的内容解析方式和含义, 定义由各模块指定(AISDK_CMD_*开头的常量)
data数据存储区域
lendata的长度
userData传回用户自定义数据，此数据是在用户发起请求是传入的
userDataLen用户自定义数据长度
extraData附加数据，如果没有返回NULL
extraDataLen附加数据长度
警告
不要在回调函数内做任何阻塞的操作。否则会阻塞其他回调。

参见各接口头文件的AISDK_CMD_*常量定义
